# 📊 Power BI Sales Dashboard

This project is an interactive sales dashboard built using Power BI to analyze global sales performance.

## 🔹 Features
- Sales trend analysis (monthly)
- Country-wise sales performance
- KPI metrics (Total Sales)
- Comparison with previous months
